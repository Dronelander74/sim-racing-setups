

\# Assetto Corsa Competizione – Reference Documentation



This folder contains \*\*official and community reference documentation\*\* for Assetto Corsa Competizione (ACC).



⚠️ \*\*Important:\*\*  

This documentation is provided \*\*for human understanding only\*\*.  

It is \*\*NOT\*\* part of the runtime contract and \*\*MUST NOT\*\* be used as a source of numeric setup values.



\---



\## Purpose of This Folder



The materials contained here are intended to:



\- Explain \*\*ACC vehicle physics concepts\*\*

\- Describe \*\*cause–effect relationships\*\* between setup parameters

\- Define \*\*valid physical ranges and behaviours\*\*

\- Support \*\*human reasoning and validation\*\*



They are useful to understand \*why\* a change works, \*\*not\*\* \*which number to use\*.



\---



\## What This Documentation MUST NOT Be Used For



The documentation in this folder \*\*MUST NOT\*\* be used to:



\- Derive numeric setup values

\- Fill missing fields in a setup

\- Generate baselines or defaults

\- Override field-manifest or car-capabilities

\- Bypass the runtime contract



Any system or process that uses these documents as a \*\*numerical source\*\* is considered \*\*non‑compliant\*\*.



\---



\## Source of Truth Hierarchy



For clarity, the project follows this strict hierarchy:



1\. \*\*field-manifest\*\*  

&#x20;  → Defines \*which fields exist\* and their structure



2\. \*\*car-capabilities\*\*  

&#x20;  → Defines \*which fields are modifiable\*



3\. \*\*Official / verified ACC setups\*\*  

&#x20;  → \*\*Only valid source of numeric values\*\*  

&#x20;  (official presets or community setups with declared provenance)



4\. \*\*This documentation\*\*  

&#x20;  → Contextual knowledge only (ranges, physics, semantics)



\---



\## Contents



Typical documents stored here include:



\- ACC Physics Notes (e.g. tyre model, bumpstop behaviour, suspension)

\- ACC Setup Guides

\- Official FAQs and patch notes

\- High‑level explanations of setup parameters



File formats are intentionally kept as \*\*PDF / DOCX\*\* and \*\*not converted\*\* into machine‑readable data.
